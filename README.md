# **Py**thon **B**ot **EX**pression (PyBEX)

A simple Interpreted Dynamic Programming Language for describing chat bot commands and behavior.

## Language Elements

### String

```
"Hello, World"
```

### Integer

```
69
420
```

### Variable

```
args
sender
```

### Funcalls

```
f()
f(g())
f(69, 420)
f("hello, world")
```
